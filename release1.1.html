Facebook Style

The model can let the user chat directly in the site, to enhance the viscosity between users and improve the site traffic. At the same time, it can provide more customized service, such as binding the existing IM, let user can chat with "site friends" but also "IM friends". Except the above features, the model also has the following features:

  1. The user can chat with "site friends" and "IM friends" in the same time.
  2. It can be showed in the several pages at the same time. It promises that it setup only one keep-alive connection with server. Saving the resource, at the same time, it ensures that these pages can also synchronize the data.
  3. Simple interface, running more stable, easy to develop and customize.
  4. Apply Client-Storage technology to save the recent chat log.
  5. Small amount of the code, the downloaded files include Javascript+CSS+Image, only 50K after GZIP.

There is no Server side for the demo, only for the front disply. Please understand that some functions cannot be seen in the demo.
